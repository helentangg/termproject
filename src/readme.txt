Project Title and Description
    This project is Plants vs. Zombies. This is a battle game where the user uses plants to fight off
    zombies. The user will be able to buy plants using sun and use them to battle the zombies. 
    The plants will be placed onto a grid in which the game will take place. Zombies will be spawned 
    onto the right end of random rows of the grid and the user loses the game once the zombies reach 
    he left end of the row. Zombies will be spawned in 3 waves, with each wave progressively getting 
    more difficult. 

Run Instructions
    Run the file main.py. 
    CMU Graphics and PIL need to be installed. CMU Graphics can be installed through the instructions
    on CS Academy. PIL can be installed using pip.

Shortcut Commands
None.
